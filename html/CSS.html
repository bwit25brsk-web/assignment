The Concept

What is CSS?

Styling the structure

CSS (Cascading Style Sheets) controls the visual presentation of HTML. It separates what a page contains from how it looks — colours, fonts, spacing, layouts, and animations.

  • Selectors target HTML elements
  • Properties set values: color: blue
  • The cascade decides which rule wins

Flexbox

Flexbox is a one-dimensional layout model. Items in a flex container arrange along a row or column, distributing space automatically and aligning with minimal code. 

.container {
           display: flex;
            gap: 1rem;
            align-items: center;
           }

Grid

CSS Grid is a two-dimensional layout system — rows and columns at the same time. It's ideal for page layouts and component grids. 

.grid {
  display: grid;
  grid-template-columns:
    repeat(3, 1fr);
  gap: 0.5rem;
}

Animations

CSS can animate properties over time using @keyframes and the animation shorthand — no JavaScript needed for many effects. 

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}
.spinner {
  animation: spin 2s linear infinite;
}

Reflection

💭 Reflection

CSS clicked for me when I stopped memorising properties and started thinking in boxes. Flexbox for one axis, Grid for two — that mental model removed most of my layout confusion. The colour picker demo was a fun way to show that CSS properties can be changed with a single JS assignment. Next step: CSS custom properties and dark mode.
