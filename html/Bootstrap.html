The Concept

What is Bootstrap?

A CSS framework

Bootstrap is a library of pre-built CSS classes and JavaScript components. Instead of writing layout CSS from scratch, you add class names like col-md-6 or btn btn-primary and Bootstrap handles the styling.

  • 12-column responsive grid system
  • Pre-styled buttons, forms, badges
  • JavaScript components: modals, dropdowns, carousels

When to use it

Bootstrap speeds up development when you need a polished, responsive UI quickly. It's less suitable when you need a highly custom design — every Bootstrap site can look the same without customisation.

  • Rapid prototyping and admin dashboards
  • Teams where not everyone knows CSS deeply
  • Projects with tight deadlines

Reflection

💭 Reflection

Bootstrap is a shortcut with trade-offs. The modal alone saved hours — it handles focus trapping and Esc-key behaviour for free. But the grid system adds many class names to HTML, and the default look is very recognisable. What I'd do differently: override CSS variables to customise the palette early, rather than fighting specificity later.
